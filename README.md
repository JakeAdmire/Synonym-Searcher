## Getting Started

To get a local copy up and running follow these simple steps:

---

### Prerequisites

* npm
```sh
npm install npm@latest -g
```

---

### Installation

1. Clone the repo
```sh
git clone https://github.com/JakeAdmire/Synonym-Searcher
```
2. Install NPM packages
```sh
cd Synonym-Searcher && npm install
```
